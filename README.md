## ihost

> host工具

* Support local create host file
* Support without delay switch host
* Support macOS

<img src="https://user-gold-cdn.xitu.io/2020/4/23/171a4e294d2118de?w=1606&h=1054&f=png&s=125334" width="500px" />


## 本地运行

``` bash
# 安装依赖
npm install

# 运行在本地 localhost:9080
npm run dev

# 打包安装包
npm run build


```
